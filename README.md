# Town Concert Hall — Event Center Website

Welcome to **Town Concert Hall**, a responsive multi-page website designed to showcase event offerings, schedules, and venue details.  
This project demonstrates modern front-end web design principles including **semantic HTML**, **responsive CSS layouts**, and **interactive UI features**.

## Project Overview

The Town Concert Hall website simulates a real-world event venue.  
It provides visitors with details about upcoming concerts, the hall’s history, and an easy way to book tickets or contact the venue.

### HTML Structure
- Semantic layout using `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, and `<footer>`.
- Logical navigation linking all pages.
- Use of headings (`h1–h6`), lists, and tables for structured content.
- Internal, external, and anchor links throughout.
- Contact form with:
  - Text input (`name`)
  - Email input
  - Textarea for messages
  - Submit button

### CSS Styling
- Consistent color palette and typography.
- Flexbox and CSS Grid layouts for responsive organization.
- Custom button and link hover effects.
- Background gradients for modern aesthetic.
- Image hover zoom in the gallery.
- Clear padding, margins, and border styling (box model).
- Includes one class selector and one ID selector example.

### Responsive Design
- Mobile-first approach using `@media` queries.
- Hamburger Menu Navigation for small screens:
  - Expands smoothly into a dropdown.
  - Uses the CSS-only checkbox toggle method.
  - Includes a smooth animation and proper positioning.
- Website adapts gracefully between desktop, tablet, and mobile screens.

## Visual Overview

- Desktop and Mobile views, responsive to adjustments in screen size

## Technologies Used
- HTML5 for structure and semantic meaning  
- CSS3 for styling, layout, and animations  
- Flexbox & Grid for responsive design  
- No JavaScript required (pure CSS interactivity)
